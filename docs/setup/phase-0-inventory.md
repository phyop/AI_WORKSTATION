# Phase 0 Inventory

Phase 0 records enough information to plan the workstation build without publishing identifying details.

## Capture

- operating system family and edition class;
- CPU, memory, storage, and GPU summary;
- power policy summary;
- time-service health;
- network posture without IP addresses or adapter identifiers;
- whether administrative access is available for later approved steps.

## Do Not Capture

- hostnames;
- account names;
- serial numbers;
- IP addresses;
- MAC addresses;
- SSH fingerprints;
- local home paths;
- raw diagnostic logs.

## Output

Use short summarized statements and neutral placeholders. The public repository should explain the engineering state, not expose the machine.
